# 课堂古文答题工具 Class Quiz PRD

## 一、产品概述

课堂古文答题工具是一个基于本地题库文件的轻量级答题网站。

老师提前准备好题库数据文件，系统自动解析题目并生成答题流程。  
学生或老师可以逐题作答，系统提供即时反馈，并在答题结束后展示统计结果与解析。

该产品主要用于课堂教学和课后练习，特别适合：

- 古文词义理解
- 阅读理解
- 语境判断题
- DSE 文言文训练

产品特点：

- 单题答题模式
- 清晰的原文语境展示
- 即时对错反馈
- 显示答题进度
- 显示答案解析
- 轻量级部署（无需后台）

第一版本重点是 **答题体验与课堂展示效果**。

---

# 二、产品目标

提供一个 **类似测评网站的课堂答题体验**。

核心目标：

1. 提供单题逐步答题体验
2. 显示即时对错反馈
3. 展示答题进度
4. 提供简单结果统计
5. 支持古文语境型题目

---

# 六、页面结构

## 6.1 首页

首页用于选择题集。

页面内容：

- 网站标题
- 题集列表
- 开始答题入口
- 后台入口（占位）

示例：

```

课堂古文练习

选择题集：

古文动词「引」语义练习
史记阅读理解
DSE文言词义训练

[进入后台]

```

点击题集进入答题流程。

---

## 6.2 答题页

答题页是核心页面。

页面布局：

```

右上角统计

第 2 / 10 题
正确 1
错误 0

---

【原文语境】

东汉·班固《汉书》
“于是延广引过归己，上甚贤之。”

---

【问题】

请判断句中“引”的意思：

A. 拉开弓箭
B. 引用典故
C. 招致 / 引起
D. 承担 / 归属于自己

```

---

## 6.3 答题逻辑

用户点击选项后：

系统立即判断答案。

### 若答案正确

- 选项变为绿色
- 显示 ✔ 标记
- 触发轻量彩带动画

显示解析：

```

解析：

“引过归己”表示把过错拉到自己身上，
因此表示承担责任。

```

---

### 若答案错误

- 选项变为红色
- 显示正确答案
- 显示解析

---

### 自动切题

显示解析约 2 秒后：

自动进入下一题。

# 十、题库数据结构

## QuizSet（题集）

字段：

- id
- title
- description
- questions

示例：

```

{
id: "ancient-chinese-yin",
title: "古文动词「引」语义理解",
description: "通过语境理解引字含义",
questions: [...]
}

```

---

## Question（题目）

字段：

- id
- context（原文语境）
- question（问题）
- options
- correctIndex
- explanation
- tags（可选）

---

### 示例题目

```

{
id: "yin-1",

context: "东汉·班固《汉书》：“于是延广引过归己，上甚贤之。”",

question: "请判断句中“引”的意思：",

options: [
"拉开弓箭",
"引用典故",
"招致 / 引起",
"承担 / 归属于自己"
],

correctIndex: 3,

explanation: "“引过归己”表示把过错拉到自己身上，因此表示承担责任。",

tags: ["古文","词义","DSE"]
}

```

# 十一、交互设计原则

## 单题模式

页面一次只展示一题。

避免信息干扰。

---

## 原文优先

古文语境必须清晰展示。

阅读体验优先。

---

## 即时反馈

点击选项后立即判断答案。

增强互动感。

---

## 自动切题

答题完成后自动进入下一题。

保证课堂节奏。

---

## 固定统计

右上角始终显示：

- 当前题号
- 总题数
- 正确数
- 错误数

---

# 十二、视觉设计原则

整体风格参考：

- 测评类网站
- 刷题网站
- 简洁教育产品

视觉要求：

- 页面留白
- 大字号题干
- 清晰按钮
- 简洁动画

动画建议：

- 勾号动画
- 小型彩带效果

避免复杂动画。
